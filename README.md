Это приложение позволяет узнать текущую погоду в выбранном городе. 

### Запуск проекта
Клонируйте этот репозиторий
`$ git clone https://github.com/Aleksandr4549/weather-app.git`

Установите зависимости с помощью 
`npm install`
или
`yarn install`

Запустите проект
`npm install start`
или
`yarn start`
